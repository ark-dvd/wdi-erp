## 1. Purpose (LOCKED)
This document defines the non-negotiable security, privacy, and data-handling principles of the WDI ERP system.

Security is a system property, not a feature.

---

## 2. Core Security Principles (LOCKED)
- Least privilege by default
- Explicit access over implicit access
- Zero-trust between modules
- Defense in depth

---

## 3. Data Classification (LOCKED)
- Public
- Internal
- Confidential
- Restricted (PII, Financial, Legal)

Each class mandates storage, access, and audit requirements.

---

## 4. Data Access Rules (LOCKED)
- All access is authenticated
- All sensitive access is authorized
- All critical access is logged

---

## 5. Privacy Doctrine (LOCKED)
User data exists solely to serve operational needs.
Secondary use is forbidden unless explicitly approved.

---

## 6. Breach & Incident Handling (LOCKED)
Incidents are assumed possible.
Preparedness is mandatory.
